# Training Planner

**Автор:** Галиев Наиль Дамирович

**Дата создания:** Апрель 2026
## Описание

Training Planner — это графическое приложение на Python для планирования тренировок. Позволяет добавлять, удалять и фильтровать тренировки по дате и типу. Данные сохраняются в JSON-файл.

## Функциональность

- Добавление тренировок (дата, тип, длительность)
- Валидация ввода (формат даты, положительная длительность)
- Фильтрация по типу тренировки и дате
- Удаление тренировок
- Сохранение/загрузка в JSON
- Простой графический интерфейс (tkinter)

## Установка и запуск

1. Убедитесь, что установлен Python 3.6+
2. Скачайте файл `training_planner.py`
3. Запустите приложение:
   ```bash
   python training_planner.py

## Сценарий: Планирование недели тренировок
## Шаг 1. Запуск приложения

Откройте программу. Вы увидите окно с полями для ввода и пустой таблицей.

## Шаг 2. Добавление первой тренировки

Дата: 2024-12-25

Тип тренировки: Бег

Длительность: 30

Нажмите «Добавить тренировку»

Результат: В таблице появилась запись: 2024-12-25 | Бег | 30

## Шаг 3. Добавление второй тренировки

Дата: 2024-12-26

Тип тренировки: Плавание

Длительность: 45

Нажмите «Добавить тренировку»

Результат: В таблице две записи.

## Шаг 4. Добавление третьей тренировки

Дата: 2024-12-27

Тип тренировки: Бег

Длительность: 40

Нажмите «Добавить тренировку»

Результат: В таблице три записи.

## Шаг 5. Фильтрация по типу «Бег»

В поле «Фильтр по типу» введите Бег

Нажмите «Применить фильтр»

Результат: Показаны только тренировки с типом «Бег» (25.12 и 27.12)

## Шаг 6. Фильтрация по дате

В поле «Фильтр по дате» введите 2024-12-26

Нажмите «Применить фильтр»

Результат: Показана только тренировка от 26.12 (Плавание)

## Шаг 7. Сброс фильтра

Нажмите «Сбросить фильтр»

Результат: Снова видны все три тренировки.

## Шаг 8. Удаление тренировки

Выберите строку с тренировкой «Плавание»

Нажмите «Удалить выбранную тренировку»

Результат: Тренировка удалена, остались только «Бег» 25.12 и 27.12

## Шаг 9. Проверка сохранения данных

Закройте приложение

Откройте снова

Результат: Все добавленные тренировки сохранились и отображаются в таблице.
